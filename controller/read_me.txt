Everything needed to get the game controller working

- Mosquitto 1.5.5 Client
Download here: https://mosquitto.org/files/binary/

- Arduino IDE 2.0.3
1. Install ESP-32 S2 Board Definition
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

2. Install IMU Board Library
Library Manager: SparkFun 9DoF IMU Breakout - ICM 20948

3. Install PubSubClient
Library Manager: PubSubClient by Nick O'Leary
